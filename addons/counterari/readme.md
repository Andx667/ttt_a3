# Konterartillerie

## Benutzung

Modul `Konterartillerie - Feindl. Artillerie` platzieren und mit allen feindlichen Artilleriegeschützen synchronisieren, die Gegenfeuer geben sollen. Modul `Konterartillerie - Freundl. Artillerie` platzieren und mit allen freundlichen Artilleriegeschützen synchronisieren, die Gegenfeuer auf sich ziehen sollen.
